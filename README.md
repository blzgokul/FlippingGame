# Memory Flip Game

## Description
This is a simple **Memory Flip Game** built with **HTML, CSS, and JavaScript**.  
The goal is to find matching pairs of cards by flipping them.

## Features
- 16 cards (8 pairs) shuffled randomly
- Click to flip cards
- Match cards to keep them flipped
- Mismatch flips back automatically
- Restart button to start a new game

## How to Play
1. Click on any card to flip it.
2. Click a second card to find its match.
3. If the cards match, they remain flipped.
4. If they don't match, they flip back after 1 second.
5. Repeat until all pairs are found.

## Technologies Used
- HTML
- CSS
- JavaScript (DOM manipulation)
